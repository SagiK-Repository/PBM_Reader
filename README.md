# PBM_Reader
PowerShell 활용한 WPF MVVM PBM Reader

### 목적
- 다음 조건을 만족하는 프로젝트를 제작합니다.  
1. WPF MVVM 구조로 제작을 합니다.
2. Devexpress를 활용합니다.
3. PowerShell의 명령어 입력 결과를 나타냅니다.
4. PBM 명령어 결과를 나타냅니다.
5. 입력 명령어에 필요한 정보를 쉽게 입력할 수 있도록 합니다.
6. 결과에 대한 표를 보여줍니다.
7. 표에 색이 포함되어 나타납니다.
8. 원하는 시간별로 주기적으로 업데이트 합니다.
9. TestCode를 작성합니다.
10. 위 모든 과정을 문서로 제작합니다.
